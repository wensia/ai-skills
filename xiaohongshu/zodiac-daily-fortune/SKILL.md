# 星座每日运势生成器

为小红书生成星座每日运势分析文案，采用四元素 + 行星象征 + 宫位触发的多维框架。

---

## 第一步：多维时间节点判断

根据日期 `{{YYYY-MM-DD}}` 判断当前属于以下哪些时间节点（可叠加）。先判断所有节点，再综合影响。在全文语义中自然体现节点气质，不直接点名分类。

### 日历节点

| 节点类型 | 判断规则 | 特征 |
|----------|----------|------|
| 年初节点 | 1月1日前后 | 新起点、重置感 |
| 月初/月末 | 每月前3天/后3天 | 开启、收尾 |
| 季度切换 | 3/6/9/12月 | 阶段性转折 |
| 普通日 | 无明显时间放大效应 | 日常调整 |

## 标题生成规则

| 要素 | 要求 |
|------|------|
| 字数 | 10-20字 |
| 必含 | "射手"或具体日期 |
| 句式 | 陈述/提示/对比式，不用疑问句 |
| 关键词 | 根据时间节点动态选择 |

**关键词风格参考**：

| 时间节点 | 推荐关键词 |
|----------|------------|
| 时间节点强时 | 觉醒/校准/临界点/转向 |
| 普通日 | 提醒/看清/调整/筛选 |
| 星座切换期 | 重启/切换/过渡 |
| 月末/季末 | 收回/总结/清理 |

**张力增强技巧**（从爆文提炼）：
- 标题：垂类关键词 + 时间/节点动作词 + 轻度好奇
- 开头钩子：第一句用观点式或短句结论，先定调再解释
- 段落节奏：短句推进，"判断 → 解释 → 微调动作"
- 情绪价值：可对号入座的感受句
- 互动引导：末段留轻问句，不强求留言

## 正文结构：四元素动态框架

正文共 5 段，每段 50-100 字。以四元素为结构骨架，根据当日星象、行星与时间节点动态判断每段侧重点。

### 第一段 | 火元素

关注行动力、动力、方向感。关联行星：太阳（核心自我）、火星（冲动）。根据当日能量判断是需要启动还是收敛。

### 第二段 | 水元素

关注情绪、关系、感受。关联行星：月亮（身体的信号、说不清的感觉）。

**塔罗圣杯意象必须体现**：情绪的容量、承载与溢出、情感的杯子。这一意象让抽象的情绪有了具体的视觉锚点，读者更容易产生共鸣。

示例：
- "情绪的杯子今天容易满溢"
- "圣杯提醒你，不是所有感受都要装进来"
- "你的情感杯子已经装了太多别人的事"

### 第三段 | 土元素

关注现实、责任、时间、金钱。关联行星：土星（逃不掉的现实）。不是制造焦虑，而是"承认现实反而轻松"。

### 第四段 | 风元素

关注沟通、边界、自由感。关联行星：水星（脑子里的声音）。主题可能是设边界，也可能是恢复沟通。

### 第五段 | 整合段

综合四元素 + 星象 + 节气。给出"调整方向"，而非结论或许愿。

**整合段必须包含**：
- 提及"第9宫"（射手座的主宫位，代表方向、意义、人生路径）
- 呼应当日星象（月相/行星能量）
- 若在节气附近，呼应节气能量

第9宫表达示例：
- "第9宫提醒你，方向需要被校准"
- "把第9宫指向的路再看清一点"
- "第9宫关心的是你真正想走的路"

## 行星使用规则

每篇选择 2-4 颗行星作为主导，不需要平均使用。用生活化语言表达，不使用专业术语。

| 行星 | 象征领域 | 隐性表达方式 |
|------|----------|--------------|
| 太阳 | 自我、目标 | "你内心真正想要的" |
| 月亮 | 情绪、直觉 | "身体的信号" |
| 水星 | 沟通、思维 | "脑子里的声音" |
| 火星 | 行动力、冲动 | "想冲的冲动" |
| 木星 | 扩张、信念 | "更大的可能" |
| 土星 | 责任、限制 | "逃不掉的现实" |

---

## 内容准则

### 强调

- "看清 — 调整 — 校准方向"的认知线
- 行星用生活化语言，不堆术语
- 宫位只点名第9宫，其余不直接说
- 语言有代入感，像在对话
- 段落主题动态判断，不机械套用

### 避免

- 具体行星度数或专业术语
- 直接预测结果（"今天会遇到真爱"）
- 煽动焦虑（"小心破财！"）
- 过度乐观的保证
- 给宏大目标（"今天适合规划人生方向"）
- 与当前日期不符的时间词
- 鸡汤式结尾
