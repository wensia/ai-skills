# 排版规则详细参考

## 智能排版（根据内容调整字号）

### 封面标题

根据文字长度动态调整字号：

| 情况 | 处理方式 |
|------|----------|
| 标题 ≤4 字 | 大字号（96-110px） |
| 标题 5-8 字 | 中等字号（72-96px） |
| 标题 >8 字 | 较小字号（56-72px） |

### 内容页正文

根据内容长度调整字号：

| 情况 | 处理方式 |
|------|----------|
| 内容 ≤150 字 | 大字号（40-44px），行高 2.0 |
| 内容 150-300 字 | 中等字号（36-40px），行高 1.9 |
| 内容 >300 字 | 较小字号（32-36px），行高 1.8 |

## 行尾排版规则

海报排版中，每行结尾应该是文字而非标点符号。这是因为标点在行尾会产生视觉上的"空洞感"，破坏整体节奏。小红书海报普遍遵循这一视觉规范。

### 换行处理

| 标点 | 处理方式 |
|------|----------|
| 逗号（，） | 逗号前内容作为一行，逗号删除或移到下一行开头 |
| 句号（。） | 句号前内容作为一行，句号删除 |
| 破折号（——） | 破折号前内容作为一行，破折号移到下一行开头（起视觉引导作用） |
| 感叹号/问号 | 符号前内容作为一行，符号删除 |

### HTML 换行示例

```html
<!-- 避免：标点在行尾 -->
<p class="content">
  射手表面上看起来，<br/>
  不在乎任何人。
</p>

<!-- 推荐：文字在行尾 -->
<p class="content">
  射手表面上看起来<br/>
  不在乎任何人
</p>
```

### 长句拆分示例

原文：`射手表面上看起来不在乎任何人，其实心里早就给你留了位置——只是嘴硬不说。`

```html
<p class="content">
  射手表面上看起来<br/>
  <span class="accent">不在乎</span>任何人<br/><br/>
  其实心里早就<br/>
  给你<span class="accent">留了位置</span><br/>
  ——只是嘴硬不说
</p>
```
