# 风格锁定系统详细参考

## 核心规则

同一套图必须使用统一风格和布局。这是因为小红书套图（滑动浏览）需要视觉连贯性，风格跳跃会让用户感觉不专业。

## 布局锁定执行流程

### 步骤 1：生成封面前确定双锁定

在生成第一张图（封面）之前，确定：
1. **风格包**：从 4 种中随机选择 1 种
2. **布局变体**：从 5 种中随机选择 1 种

批量生成多条记录时，确保每条记录使用不同的风格组合，避免视觉重复。

**随机选择公式**：
```
风格包编号 = (记录索引 % 4) + 1
布局变体编号 = (记录索引 % 5) + 1
```

**风格包对应表**：

| 编号 | 风格包 |
|-----|-------|
| 1 | 经典强调 |
| 2 | 简约边框 |
| 3 | 杂志双线 |
| 4 | 艺术镂空 |

**布局变体对应表**：

| 编号 | 变体 | 名称 | 特征 | 关键CSS类 |
|-----|------|------|------|---------|
| 1 | **A** | 色块标题居中式 | 居中对称，色块标题 | `.main-a`, `.title-block` |
| 2 | **B** | 杂志章节式 | 左对齐，PART标签 | `.main-b`, `.part-label` |
| 3 | **C** | 数字序号引导式 | 大淡色数字背景 | `.main-c`, `.big-number` |
| 4 | **D** | 引用突出式 | 大引号开头 | `.main-d`, `.big-quote` |
| 5 | **E** | 分栏对比式 | 左侧标签+右侧内容 | `.main-e`, `.left-column` |

**布局变体参考建议**（仅供参考，随机优先）：

| 内容类型 | 建议布局 | 原因 |
|---------|---------|------|
| 年运势/月运势 | B | PART标签适合分章节展示 |
| 每日运势 | A | 简洁明了，突出单日主题 |
| 情感/心理/感悟 | D | 大引号适合金句和感悟 |
| 多话题 | E | 左标签右内容，清晰分类 |
| 清单/指南/规则 | C | 数字引导适合列表内容 |

### 步骤 2：封面添加双锁定注释

封面 HTML 包含完整的双锁定标记：
```html
<!-- [STYLE LOCK: 简约边框] [LAYOUT LOCK: B] -->
<!-- 本套图所有内容页必须使用此风格和布局 -->
```

### 步骤 3：每张内容页继承锁定配置

生成每张内容页时：
1. 先读取封面的 `[STYLE LOCK]` 和 `[LAYOUT LOCK]` 注释
2. 使用完全相同的风格包和布局变体
3. 在内容页 HTML 中添加相同的双锁定注释

配置示例：
```
套图配置：
- 风格包：简约边框 ← 随机选择（编号2）
- 布局变体：B（杂志章节式）← 随机选择（编号2）
```
