# 性格独白风 - AI 生成规范

## 一、模板概述

| 属性 | 值 |
|------|-----|
| 风格 | 温暖治愈、深度解读、书信感 |
| 语气 | 第一人称独白、亲切口语化 |
| 适合话题 | 性格分析、情感表达、内心独白、减压方式、社交模式 |
| 不适合 | 星座配对、互动投票、测试题、运势预测 |

### 设计特征
- PART 标签分章节
- 左边框引用块
- 渐变暖色背景 (#FAF6F1 → #F0E6D9)
- 强调色 `#C4653A`

## 二、内容格式规范

### 2.1 封面 (cover) - 爆款公式

> ⚠️ **核心原则**：封面是点击率的关键，必须**极简+强冲突+制造好奇**

#### 封面结构（从上到下）

```
┌─────────────────────────────────┐
│                                 │
│         [副标题]                │  ← 顶部小字，情绪标签
│                                 │
│       [主标题第一行]            │  ← 大字，固定2行
│       [主标题第二行]            │  ← 大字高亮，核心词
│            ───                  │  ← 分隔线
│         [点缀语]                │  ← 补充说明
│                                 │
└─────────────────────────────────┘
```

#### 字段规范

| 字段 | 字数范围 | 格式要求 | 示例 |
|------|----------|----------|------|
| **副标题** | 2～6字 | 情绪标签/痛点词，制造代入感 | `扎心了` `反差萌真相` `真相警告` |
| **主标题** | **固定2行**，共**8-12字** | 必须包含星座名，第二行含高亮词 | 见下方规则 |
| **点缀语** | 4～10字 | 转折/金句/悬念，补充主标题 | `粘起来不要命` `嘴上说没事` |

#### 主标题规则（核心要求）

> ⚠️ **三个硬性要求**：
> 1. **必须包含星座名称**（射手座/天秤座/双子座等）
> 2. **固定2行**，总字数 **8-12字**（≥8且<13）
> 3. **必须吸引点击**（制造悬念/反差/共鸣）

| 换行模式 | 第一行 | 第二行（含高亮） | 总字数 |
|----------|--------|------------------|--------|
| **反差模式** | 射手座的苦 | 扛得住【心寒】扛不住 | 12字 |
| **悬念模式** | 射手座为什么 | 【突然消失】 | 9字 |
| **共鸣模式** | 射手座的 | 【依赖】有多可怕 | 10字 |
| **榜单模式** | 最【上头】的 | 星座TOP3 | 8字 |
| **揭秘模式** | 射手座暧昧期 | 【试探方式】 | 10字 |

**换行原则**：
1. **星座名必须出现**在第一行或第二行
2. 第二行必须包含**高亮核心词**（用【】标记）
3. 两行形成**语义张力**（对比/递进/转折/悬念）
4. 总字数8-12字，简洁有力吸引点击

#### 爆款标题公式

| 公式 | 第一行 | 第二行（高亮） | 完整示例 |
|------|--------|----------------|----------|
| **反差公式** | XX座的【A】 | 其实是【B】 | `射手的冷漠` / `其实是【保护】` |
| **悬念公式** | XX座为什么 | 【现象】 | `射手为什么会` / `【突然消失】` |
| **共鸣公式** | XX座的 | 【痛点】时刻 | `射手座的` / `【崩溃】时刻` |
| **能力公式** | XX座什么X都能X | 却X不住【Y】 | `射手什么苦都能吃` / `却扛不住【心寒】` |
| **榜单公式** | 最【特征】的 | 星座TOP3 | `最会【冷暴力】的` / `星座TOP3` |

#### 吸引点击的心理钩子

1. **身份认同**：让目标用户觉得"这说的就是我"
2. **情感共鸣**：戳中痛点、说出心声
3. **好奇悬念**：制造信息差，想知道答案
4. **反差冲突**：打破刻板印象，出乎意料
5. **稀缺独特**：只有XX座才懂的、专属的

#### 高亮词选择（封面只用1个）

- **只高亮1个核心词**，必须出现在主标题第二行
- 高亮词是整个内容的**情感爆点**或**反差点**
- 示例：`【生理性依赖】` `【心寒】` `【消失】`

#### 封面固定排版参数

| 元素 | 字体 | 字号 | 字间距 | 对齐 | Y坐标 |
|------|------|------|--------|------|-------|
| 副标题 | Noto Serif SC | 32px | 6px | 居中 | 480 |
| 主标题第一行 | Noto Serif SC | 80px | 8px | 居中 | 620 |
| 主标题第二行 | Noto Serif SC | 80px | 8px | 居中 | 730 |
| 分隔线 | - | - | - | 居中 | 800 |
| 点缀语 | Noto Serif SC | 36px | 4px | 居中 | 890 |

#### 封面示例

**示例1：反差模式** ✅ 12字
```
副标题: 扎心了            (3字)
主标题第一行: 射手座的苦        (5字)
主标题第二行: 扛得住【心寒】扛不住 (7字, 高亮)
点缀语: 懂事的人最委屈    (6字)
```

**示例2：共鸣模式** ✅ 10字
```
副标题: 反差萌真相        (5字)
主标题第一行: 射手座的          (4字)
主标题第二行: 【依赖】有多可怕  (6字, 高亮)
点缀语: 粘起来不要命      (6字)
```

**示例3：榜单模式** ✅ 8字
```
副标题: 星座榜单          (4字)
主标题第一行: 最【上头】的      (4字, 高亮)
主标题第二行: 星座TOP3          (4字)
点缀语: 爱得快也走得快    (6字)
```

**示例4：揭秘模式** ✅ 10字
```

主标题第一行: 射手座暧昧期      (6字)
主标题第二行: 【试探方式】      (4字, 高亮)
点缀语: 小心机大揭秘      (6字)
```

### 2.2 内页 (page) × 5-8 页

> ⚠️ **内页数量**：最少5页，最多8页（根据内容丰富程度调整）

| 字段 | 字数范围 | 格式要求 | 高亮规则 |
|------|----------|----------|----------|
| 章节标签 | 固定 | PART 01/02/03... | 无 |
| 章节标题 | 4-8字 | 名词短语或动词短语 | 无 |
| 正文 | **4-6行（硬性要求）** | 每行5-15字 | 至少1组【呼应词】 |
| 引用 | 8-20字 | 金句，双引号包裹 | 可选0-1个 |

> 🚨 **正文行数硬性要求**：
> - **最少4行，最多6行**
> - 少于4行会导致页面过于空旷，影响视觉效果
> - 如果原始内容不足4行，必须扩写补充相关内容
> - 扩写方向：添加具体场景、补充情绪描写、增加对比或转折

**内页数量指南**：
| 内容类型 | 建议页数 | 说明 |
|----------|----------|------|
| 简单话题 | 5页 | 如：穿搭风格、电影类型 |
| 标准话题 | 6页 | 如：性格特质、恋爱表现 |
| 丰富话题 | 7-8页 | 如：TOP榜单、多维度分析 |

**内页高亮原则**：
- 每段正文必须有**至少1组高亮词**（2个词形成呼应）
- 两个词形成语义呼应：
  - **对比**：【直觉】vs【脑子】
  - **递进**：【想】→【做】
  - **因果**：因为【感觉】所以【选择】
  - **转折**：表面【冷】内心【热】
- 高亮词**不要连续**，中间需有普通文字间隔

**⚠️ 内容不足4行时的扩写策略**：

| 原始内容 | 扩写方向 | 扩写示例 |
|----------|----------|----------|
| 2行核心观点 | 添加场景描写 | 原：内向版的INFP / 安静时像换了个人 → 扩写为4行 |
| 3行内容 | 补充情绪/细节 | 添加"为什么"或"具体表现" |

**扩写示例**：

❌ **错误（只有2行，太空）**：
```
内向版的【INFP】
安静时像换了个人
```

✅ **正确（扩写到4行）**：
```
有时候也会是
内向版的【INFP】
安静时像换了个人
【独处】才是真正的充电
```

✅ **正确（扩写到5行）**：
```
但射手也可能是
内向版的【INFP】
安静时像换了个人
需要【独处】来充电
热闹过后更想一个人待着
```

### 2.3 结尾页 (summary) - 居中布局

**结尾页原则**：
- **所有文字居中**（text-anchor="middle"）
- 总结正文与封面主题呼应
- 高亮词承接封面的情感主线
- 祝福语是情感升华，语气温暖治愈

## 三、文案风格指南

### 3.1 语言风格
- **视角**：第一人称（"我们XX座"、"射手就是这样"）
- **语气**：口语化、亲切、有共鸣感
- **句式**：短句为主，节奏感强
- **禁忌**：避免书面语、长句、生硬描述

### 3.2 话题适配示例

| 话题类型 | 封面主标题示例 | 内页章节示例 |
|----------|---------------|-------------|
| 性格特质 | 射手座什么【苦】都能吃\n却扛不住心寒 | 表面乐观、内心敏感、习惯独撑 |
| 做决定 | 射手座做【决定】\n来不及想 | 选择的时候、犹豫的时候、果断的瞬间 |
| 减压方式 | 射手座的【减压】\n靠走 | 累了就走、一个人待着、不想解释 |
| 社交模式 | 射手座的【社交】\n看心情 | 聊天的时候、沉默的时候、独处的时候 |
| 恋爱 | 射手座【谈恋爱】\n一点就燃 | 喜欢的信号、暧昧期、确认关系 |

### 3.3 高亮词选择策略

| 呼应类型 | 说明 | 示例词组 |
|----------|------|----------|
| 表里对比 | 表象 vs 本质 | 【乐观】vs【敏感】、【洒脱】vs【细腻】 |
| 能力对比 | 能做 vs 不能做 | 【苦】vs【委屈】、【扛】vs【心寒】 |
| 行为对比 | 外在 vs 内在 | 【说】vs【想】、【走】vs【留】 |
| 递进关系 | 从浅到深 | 【累】→【走】→【好】 |
| 因果关系 | 原因 → 结果 | 【懂事】→【温柔以待】 |

### 5.3 高亮词解析

**使用【】标记高亮词，渲染时转换为 SVG tspan：**

```python
import re

def parse_highlight(text: str) -> str:
    """将【词】转换为 SVG tspan 高亮"""
    def replace(m):
        return f'<tspan fill="#C4653A">{m.group(1)}</tspan>'
    return re.sub(r'【([^】]+)】', replace, text)

# 示例
# 输入: "其实内心【敏感】得要命"
# 输出: '其实内心<tspan fill="#C4653A">敏感</tspan>得要命'
```

## 六、检查清单

### 封面检查（严格执行）
- [ ] **主标题必须包含星座名**：射手座/天秤座/双子座等
- [ ] **主标题必须2行**：不能是1行或3行
- [ ] **主标题总字数8-12字**：第一行+第二行≥8且<13
- [ ] **副标题**：2～6字
- [ ] **点缀语**：4～10字
- [ ] **只有1个高亮词**：出现在主标题第二行
- [ ] **吸引点击**：有悬念/反差/共鸣元素

### 内页检查（严格执行）
- [ ] 🚨 **正文行数：必须4-6行**（少于4行必须扩写，这是硬性要求！）
