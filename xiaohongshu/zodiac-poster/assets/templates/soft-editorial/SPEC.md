# 温柔杂志风 - AI 生成规范

## 一、模板概述

| 属性 | 值 |
|------|-----|
| 风格 | 温柔有态度、杂志感、清晰层次 |
| 语气 | 情感共鸣、轻微调侃、治愈系 |
| 适合话题 | 星座情劫、TOP排名、情感解读、星座配对 |
| 不适合 | 测试题、互动投票、运势预测 |

### 设计特征
- 单栏居中布局
- 超大情感化标题 (128px+)
- 磨砂颗粒质感 (Grain Texture)
- 奶油色暖调背景
- TOP1 页面使用深色模式突出
- 星座符号装饰性使用

## 三、页面结构

### 3.1 封面 (Cover)

| 元素 | 规格 | 说明 |
|------|------|------|
| 顶部色条 | 高度 8px | 强调色 |
| 小标签 | 背景色块 | "XX座专属" |
| 主标题 | 128px, Noto Serif SC | 多行，带高亮字 |
| 副标题 | 24px | 情感共鸣语 |
| 星座符号 | 40px | 分隔装饰 |
| 滑动提示 | 底部 | "向左滑动 →" |

**内容示例**：
```
小标签: 射手座专属
主标题: 最容易/栽的/3个星座
副标题: 不是不会爱/是爱得太深
```

### 3.2 引入页 (Page1)

| 元素 | 规格 |
|------|------|
| 页码圆 | 左上，橙色背景 |
| 小标题 | 24px，tracking 宽 |
| 大标题 | 160px，情感词 |
| 正文 | 3行，40px，带高亮 |
| 引言框 | 深色背景，斜体 |

**内容示例**：
```
小标题: 射手的

正文: 射手不轻易动心/但一旦栽了/就栽得很彻底

```

### 3.3 排名页 (TOP 3/2)

|------|------|

| TOP标签 | 左上或右上 |
| 星座符号 | 100px |

## 七、内容生成规范

### 7.1 话题：星座情劫 TOP 榜

**输入示例**：
```yaml
主星座: 射手座
话题: 最容易栽的3个星座
TOP3: 摩羯座 - 冷淡
TOP2: 双鱼座 - 深情
TOP1: 天蝎座 - 控制欲
```

**输出页面**：
1. 封面 - 情感冲击标题
2. 引入页 - 设定情感基调
3. TOP3 页 - 摩羯座分析
4. TOP2 页 - 双鱼座分析
5. TOP1 页 - 天蝎座（深色突出）
6. 成长页 - 正向升华
7. 结尾页 - 祝福收尾

### 7.2 高亮词规则

- 每页 1-2 个高亮词
- 使用 `<span style={{ color: colors.orange }}>词</span>` 格式
- 高亮词应为情感核心词或转折词

**示例**：
- "就栽得**很彻底**"
- "最后成了**渣**"
- "就是**断不掉**"
